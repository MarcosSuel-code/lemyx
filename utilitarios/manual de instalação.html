📄 Manual de Instalação e Uso — Sistema ControlSist

🔧 Requisitos

  • ✔️ PHP 7.4 ou superior
  • ✔️ MySQL ou MariaDB
  • ✔️ Servidor local (XAMPP, WAMP, Laragon, etc.)

🗂️ Instalação do Servidor Local

  1. Baixe e instale:
    • XAMPP
    • WAMP
    • Laragon (Recomendado)
  2. Abra o painel e inicie:
    • ✅ Apache
    • ✅ MySQL

📦 Instalação do Sistema

  1. Copie a pasta ControlSist.
  2. Cole na pasta do servidor local:
    • XAMPP: C:\xampp\htdocs
    • WAMP: C:\wamp\www
    • Laragon: C:\laragon\www

🗄️ Configuração do Banco de Dados

  1. Acesse phpMyAdmin: http://localhost/phpmyadmin
  2. Clique em "Novo" → Crie o banco: controlsist
  3. Vá na aba "Importar" → Escolha o arquivo sql_banco.sql
  4. Clique em "Executar" ✔️

🔗 Ajustar Conexão com o Banco

Abra o arquivo ControlSist/config/database.php e configure:

<?php
$host = "localhost";
$dbname = "controlsist";
$username = "root";
$password = "";

try {
    $pdo = new PDO("mysql:host=$host;dbname=$dbname", $username, $password);
    $pdo->setAttribute(PDO::ATTR_ERRMODE, PDO::ERRMODE_EXCEPTION);
} catch (PDOException $e) {
    die("Erro na conexão com o banco de dados: " . $e->getMessage());
}
?>

🔐 Login no Sistema

No navegador, acesse:

http://localhost/ControlSist

Use um dos usuários cadastrados no banco (ver tabela usuarios no phpMyAdmin).

⚙️ Funcionalidades

  • ✔️ Cadastro de produtos, fornecedores e categorias
  • ✔️ Controle de estoque
  • ✔️ Entrada e saída de mercadorias
  • ✔️ Relatórios e filtros por período
  • ✔️ Dashboard com gráficos e alertas de estoque baixo
  • ✔️ Sistema de login com níveis de acesso (admin e usuário)

🌐 Acesso pela Rede Local

  1. No PC servidor, abra o CMD e digite: ipconfig
  2. Anote o IP local, exemplo: 192.168.1.10
  3. Nos outros PCs da rede, acesse: http://192.168.1.10/ControlSist

☁️ Publicação na Internet (Opcional)

Contrate um servidor com suporte a PHP + MySQL.

Suba os arquivos via FTP e importe o banco pelo painel do host.

🧠 Dicas Importantes

  • 🔑 Altere as senhas padrão após a instalação.
  • 📤 Faça backups regulares do banco de dados.
  • 🚀 Para melhorias futuras, pode implementar exportação em PDF, notificações, gráficos adicionais e backups automáticos.
