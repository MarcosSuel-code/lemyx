🔍 Checklist do Sistema
🎯 Funcionalidades Confirmadas:
✅ Login e Senha (com 2 usuários iniciais: admin e padrão)

✅ Dashboard com resumo (produtos, estoque, últimas entradas e saídas)

✅ Cadastro de Fornecedores

✅ Cadastro de Produtos (com categoria e fornecedor)

✅ Cadastro de Categorias

✅ Cadastro de Tipos de Saída (ex.: venda, perda, devolução)

✅ Entrada de Mercadorias (adiciona ao estoque)

✅ Saída de Mercadorias (diminui do estoque)

✅ Relatórios (com filtros de datas para entradas e saídas)

✅ Edição e Exclusão de Registros (em todos os cadastros)

✅ Controle de Estoque

✅ Permissões de Usuário (admin faz tudo, padrão tem acesso restrito)

✅ Design Responsivo com CSS moderno

✅ Manual de Uso incluso

✅ Script SQL completo para criação do banco de dados

📦 Estrutura do Projeto:
Frontend em HTML + CSS + PHP (com Bootstrap simples customizado no CSS)

Backend todo feito em PHP puro com PDO para conexão segura ao MySQL

Banco de dados MySQL

📊 Relatórios gráficos interativos (barras, pizza, linhas)

📤 Exportação de relatórios em PDF e Excel

🔔 Sistema de notificações automáticas:
→ Estoque baixo, estoque zerado, vencimentos (se desejar)

📱 Layout mobile aprimorado:
→ Interface responsiva, parecida com aplicativo

🧠 Backup automático do banco de dados:
→ Botão para gerar e baixar backup .sql

🔑 Sistema avançado de permissões:
